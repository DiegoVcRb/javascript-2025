Javascript

Javascript es un lenguaje de programacion dinamico y de alto nivel para crear interfaces interactivas y dinamicas

Es uno de los tres pilares fundamentales del desarrollo web

  • HTML -> Define la estructura del contenido (elementos que aparecen en la pagina)
  • CSS -> Define la apariencia (colores, tamaÃ±os, estilos)
  • Javascript (JS) -> Define el comortamiento (lo que hace la pagina o sus elementos.)

Caracteristicas de Javascript

  1. Lenguaje Interpretado -> No necesitamos compilarlo. El navegador lo comprende directamente.
  2. Basado en objetos -> Maneja objetos como document, window, arrays, date, class.
  3. Dinamico -> Podemos modificar el contenido en tiempo real (sin recargar)
  4. Versatil y de facil aprendizaje -> Aunque nacio para la web, se usa tambien en servidores, apps moviles, videojuegos y hoy en dia tambien IA
